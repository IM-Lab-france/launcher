Guide d'utilisation

Introduction

Cette application vous permet de gérer vos signets en ligne de manière simple et intuitive. Suivez ce guide pour apprendre à ajouter des sections, des signets, personnaliser l'interface et bien plus encore.

Mode édition

Le mode édition permet de déplacer, renommer et supprimer des sections et des signets.

  • Activez le Mode Édition en cliquant sur l'icône ✏️ dans le menu.
  • Déplacez les sections et les signets par glisser-déposer.
  • Pour renommer ou supprimer un élément, utilisez les icônes correspondantes.

Ajouter une section

Pour organiser vos signets, vous pouvez ajouter des sections en utilisant le bouton Nouvelle Section dans la barre de menu.

  • Appuyez sur Nouvelle Section pour ouvrir une fenêtre modale où vous pourrez saisir le nom de la section.
  • Une fois le nom saisi, cliquez sur Ajouter pour créer la section.

Ajouter un signet

Pour ajouter un signet dans une section, cliquez sur l'icône ➕ à côté de la section concernée.

  • Entrez le nom du signet et l'URL du site dans la fenêtre modale.
  • Cliquez sur Ajouter le lien pour enregistrer le signet dans la section choisie.

Changer la langue

Vous pouvez modifier la langue de l'interface en utilisant le sélecteur de langue dans le menu burger.

Personnaliser le fond d'écran

Pour personnaliser l'arrière-plan, cliquez sur Personnaliser le fond d'écran dans le menu. Vous pouvez choisir une image parmi celles disponibles.

Télécharger l'application

Pour télécharger l'application, cliquez sur le bouton ci-dessous :

Télécharger Launcher.exe

Attention : Si Windows Defender empêche l'exécution, procédez comme suit pour autoriser l'installation :

  • Après le téléchargement, cliquez avec le bouton droit et sélectionnez "Propriétés".
  • Dans l'onglet "Général", cochez "Débloquer" en bas, puis cliquez sur "Appliquer".
  • Lancez ensuite le programme normalement. Si une alerte Windows Defender s'affiche, sélectionnez "Informations supplémentaires", puis cliquez sur "Exécuter quand même".

Vous pouvez désormais afficher vos pages web avec un maximum d'espace pour le contenu
